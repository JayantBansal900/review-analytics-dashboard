# 📊 Review Analytics Dashboard

## 📌 Overview
This is a UI dashboard designed for ecommerce brands to analyze thousands of customer reviews across platforms.

## 💡 Features
- Total reviews count
- Positive vs Negative sentiment
- Platform filter (Amazon, Flipkart, Myntra)
- Simple sentiment chart
- Recent reviews display

## 🛠 Tech
- HTML
- CSS
- JavaScript

## 🎯 Purpose
This was built as part of a product-thinking assignment to design a scalable review analytics system.

## 🚀 How to Run
Open `index.html` in browser
